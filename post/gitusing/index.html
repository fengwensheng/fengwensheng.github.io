Git使用

2020-07-05

记录一下常用的Git命令和操作流。

1.最常用

git clone targetUrl

clone远程项目到本地。clone时会一并将项目最外层的文件夹一并拉下来，不用担心会散落一地。

2.当前所在分支

git branch

3.历史提交记录

git log
git log –oneline

使用 git log --oneline 查看起来更友好一些，去掉了Author和Date。Commit ID也只是显示前7位，使得一行放一条记录。

4.本地项目如何回到历史提交版本

git reset –hard commitId

这时就可以配合查看远程服务器上的历史提交记录的命令一起使用了。
